# Slides for All Days Devops 2022

## Build with dagger

1. Start the devcontainer with vscode ([see devcontainer](https://code.visualstudio.com/docs/remote/containers))
2. Run dagger:

    ```bash
    dagger do build
    ```

3. Check the folder `/public`

    ```txt
    ./public/
    ├── addo2022.pdf
    ├── addo2022.pptx
    ├── assets
    └── index.html
    ```
